# EduGen AI — Custom Content Generator

A web-based AI-powered educational content generation system that automates the creation of study notes, summaries, flashcards, and interactive quizzes tailored to different educational levels.

## Features
- Study Notes Generator with structured key concepts and definitions
- Summary Generator for quick revision
- Flashcard Generator for memorisation
- Interactive Quiz with automated scoring, memo, and explanations
- PDF Export of quiz results and study materials
- Supports Primary, Secondary, and Tertiary educational levels
- Prompt template engine with grade-level adaptations
- Generation history and usage statistics tracking

## Tech Stack
HTML | CSS | JavaScript | Replit AI

## Performance
Average generation time: 29.8 seconds | 30,968 tokens used across 15 generations
